# Map_me

# Android App with Map Implementation:
 - You can make Markers 
 - You can edit your Markers 
 - Find your location
 
 # Technologies used:
 - Clean Architecture
 - Flow Coroutines for Async job
 - Room for storage datas
 - Devided by modules
 - Google Map API
